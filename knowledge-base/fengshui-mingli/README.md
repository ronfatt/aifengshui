# AI 风水命理知识库

这个目录用于存放和整理 AI 风水命理师的内部参考资料。

## 目录结构

- `raw/`：原始资料上传区。可以放 PDF、Word、PPT、图片等文件。此目录不会提交到 Git。
- `processed/extracted/`：系统从可读取文件中抽取出来的文字。
- `processed/structured/`：人工整理后的结构化判断框架。
- `indexes/`：知识地图与分类索引。
- `prompts/`：AI 使用知识库时的提示规则。
- `notes/`：处理报告与待办说明。

## 使用方式

1. 把新资料放入：

```bash
knowledge-base/fengshui-mingli/raw/
```

2. 重新生成知识索引：

```bash
npm run knowledge:build
```

3. 重启网站服务。AI 风水命理师、综合报告、一般报告会自动读取新的知识资料。

## 重要说明

- 原始书籍、图片和课程资料不建议提交到 Git，避免体积过大和版权风险。
- 目前脚本可直接读取 Word、PPT、文字文件；扫描版 PDF 与图片需要 OCR 后才能成为高质量知识。
- AI 回答时只会吸收判断逻辑，不应长篇复制原文。
- 资料越结构化，AI 判断越稳定。建议后续把知识整理为：
  - 梅花易数：起卦、体用、生克、互卦、变卦、断事模板
  - 紫微斗数：十二宫、主星、四化、大限、流年
  - 八字命理：五行、十神、日主、用神、大运流年
  - 数字命理：生命数字、个人年、姓名能量、执行节奏
  - 行为风水：方位、颜色、空间整理、通关行为、产品建议

## 线上使用边界

所有输出必须保留文化参考、自我觉察和个人规划定位，不构成金融、法律、医疗或其他专业建议。
